# Changelog

## 0.1.0 - 2026-02-13
- Initial release of mcp-firewall sidecar proxy for MCP traffic.
- Policy engine with allow/deny by tool, path, origin, signature, and body size.
- Audit JSONL logging for every decision and upstream outcome.
- CLI with `run`, `doctor`, and `demo` commands.
- CI workflows for lint/test, CodeQL, and release build scaffolding.
